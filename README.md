## Usage

``` bash
# Install dependencies
npm install

# Build, watch for changes and debug the application
tns debug <platform>

# Build for production
tns build <platform> --env.production

```

Démarrage :

``` bash
# Preview on device
tns preview

# Build, watch for changes and run the application
tns run
```
